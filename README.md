## Create. Animate. Publish.

**ICONIQ Motion Studio™** is the ICONIQ LABS™ video automation and AI-assisted production platform direction.

It turns ideas, scripts, reference videos, product concepts, brand briefs, and creative prompts into repeatable video production workflows for short-form content, faceless channels, product promos, educational explainers, cinematic montage, social media assets, and branded creator packages.

This repository is being adapted from the original **OpenMontage** project direction into an ICONIQ-branded product and service ecosystem. The license remains **GNU AGPLv3**. See [`LICENSE`](LICENSE) and [`NOTICE-ICONIQ.md`](NOTICE-ICONIQ.md).

---

## What It Does

ICONIQ Motion Studio™ is designed to help an AI coding assistant operate like a video production studio. A user can describe the desired output in plain language while the system coordinates research, scripting, asset generation, media sourcing, editing, captioning, audio, and final composition.

Core production paths include:

- AI-assisted short-form video creation
- Product promo videos for digital and physical products
- Faceless YouTube Shorts, TikTok, and Instagram Reels workflows
- Image-based animated videos
- Real-footage documentary montage workflows
- Captioned social clips
- Explainer videos
- Brand launch videos
- Repeatable creator workflow templates

## Quick Start

### Prerequisites

- **Python 3.10+**
- **FFmpeg**
- **Node.js 18+**
- **An AI coding assistant** such as Claude Code, Cursor, Copilot, Windsurf, or Codex

### Install & Run

```bash
git clone https://github.com/JOJO-ICONIQ/OpenMontage.git
cd OpenMontage
make setup
```

Open the project in your AI coding assistant and describe the video you want:

```text
Make a 45-second product promo video for a luxury digital planner brand. Use elegant black, midnight blue, gold, and diamond-inspired visual language. Include captions and a strong CTA.
```

Or:

```text
Create a 60-second faceless YouTube Short about how AI can help small business owners create content faster. Include narration, captions, and a clean branded visual style.
```

Create a 30-second Etsy product promo for a luxury digital printable called the CEO Daily Planner. Use a premium black, gold, and midnight blue visual style. Show planner pages, productivity language, and a final CTA: Plan Beautifully. Live Intentionally.
